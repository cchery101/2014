# CSSConf Australia 2014 Site

http://2014.cssconf.com.au/

## Dependencies

* Node
* Sass (`gem install sass`)

## Installing

* `npm install -g grunt-cli`
* `npm install`

## Building

* `grunt connect watch`
* `open http://localhost:3333/`

## Releasing

* `grunt release`
